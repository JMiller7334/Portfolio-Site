API Weather App

Android

A simplistic app that utilizes API services to get the weather in either Celsius or Fahrenheit.

Written in Kotlin using an MVVM (Model-View-ViewModel) design pattern this app fetches the weather by retrieving the city name of the zip code entered by the user via API; afterwards it leverages a second API to fetch the weather for corresponding town or city.
